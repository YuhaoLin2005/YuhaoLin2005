## What I Build

I make AI agents reliable. Most AI assistants drift — they forget rules, can't verify their own output, and have no memory across sessions. I build lightweight systems that wrap around LLMs to give them stable identity, self-verification, and causal feedback loops.

**My approach:** compact self-model → attention routing table → mechanical enforcement hooks → closed-loop regeneration. No model training. No weight access. Just prompt architecture and filesystem state machines.
